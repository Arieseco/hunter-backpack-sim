# 要件定義書

**プロジェクト名:** TheHunter: Call of the Wild バックパックシミュレーター  
**作成日:** 2026-04-19  
**ステータス:** 初版

---

## 1. プロジェクト概要

### 1.1 背景

「The Hunter: Call of the Wild」はプレイヤーが狩猟を行うゲームである。プレイヤーは銃器・弾薬・テント・望遠鏡・呼び笛などを携行して狩猟を行うが、**所持できる合計重量に上限**があり、その範囲内でアイテムを揃える必要がある。

現状、プレイヤーは手動で重量を計算しながら装備を選択しており、最適な装備構成の検討に手間がかかっている。

### 1.2 目的

- 携行アイテムの重量シミュレーションを簡単に行えるWebアプリを提供する
- 銃器・アイテム情報を一覧で参照できるデータベースを提供する
- シミュレーション結果をURLで共有・保存できる機能を提供する

### 1.3 対象ユーザー

「The Hunter: Call of the Wild」のプレイヤー

### 2.2 重量シミュレータ（メイン機能）

#### 2.2.1 重量計算ルール

| 条件 | 容量 |
|---|---|
| 初期状態 | 20 kg |
| 荷運びラバ スキルON | 20 × 1.15 = 23 kg |
| バックパック（小）装備 | 基本容量 + 3 kg |
| バックパック（中）装備 | 基本容量 + 6 kg |
| バックパック（大）装備 | 基本容量 + 9 kg |

- バックパックは小・中・大のいずれか1つのみ装備可能
- 荷運びラバのボーナスはバックパックのボーナスには影響しない
- 例: 荷運びラバON + バックパック大 = 23 + 9 = **32 kg**

#### 2.2.2 機能詳細

- **アイテムリスト（左パネル）**
  - 全アイテム（銃器・弾薬・その他）を一覧表示
  - テキスト検索フィルタ
  - カテゴリフィルタ（銃器種別・アイテム種別）
  - アイテムカードをホバーで説明ツールチップ表示
  - ドラッグ&ドロップで装備中パネルへ追加

- **装備中パネル（右パネル）**
  - 現在の合計重量 / 上限容量をプログレスバーで表示
  - 重量超過時は赤色で警告表示
  - 荷運びラバスキルのON/OFFトグル
  - バックパック選択（なし / 小 / 中 / 大）
  - 装備中アイテムの一覧（個別削除可・一括クリア可）
  - シミュレーション名（任意入力）
  - 保存ボタン → UUID付きURLを発行してDBに保存

- **狩猟区セレクタ（上部）**
  - 狩猟区を選択すると、その地域に出没する動物と適正狩猟クラスを表示
  - 装備選択の参考情報として利用

#### 2.2.3 保存・共有機能

- ログイン不要で保存可能
- 保存時にUUID付きURLを生成: `/simulator/{uuid}`
- URLにアクセスすると保存時の状態を復元・表示
- URLをブックマーク・共有して再利用可能

### 2.3 銃器一覧

- 銃器名・対応クラス・重量を一覧テーブルで表示
- テキスト検索フィルタ
- 詳細ページ: 銃器名・重量・コメント・対応弾薬一覧（弾薬名・重量・適正クラス）
- マズルローダー（`muzzleloader`）はDBで管理するが、専用一覧ページなし。シミュレーターのアイテムリストからアクセス可能。

### 2.4 アイテム一覧

- アイテム名・重量・説明を一覧テーブルで表示
- テキスト検索フィルタ
- カテゴリ別ページ（呼び笛 / 匂いアイテム / 装備 / 構造物）

## 3. 非機能要件

### 3.1 ユーザー認証

- **不要**。ログインなしで全機能を利用可能

### 3.2 データ管理

- ゲームデータ（銃器・アイテム等）はSupabase（PostgreSQL）で管理
- データはSQL（seed.sql）で初期投入し、Supabase Studioから編集可能

### 3.3 セキュリティ

- Row Level Security（RLS）を全テーブルに適用
- ゲームデータテーブル: 読み取り専用（クライアントからの書き込み不可）
- simulationsテーブル: 挿入・読み取りのみ許可（全件リスト取得不可）

### 3.4 パフォーマンス

- 全ページをサーバーサイドレンダリング（SSR）で配信
- Supabase無料プランで運用可能な規模

## 4. データ要件

### 4.1 管理データ

| データ | 内容 |
|---|---|
| 銃器 | 名称・種別（ライフル/ショットガン/ハンドガン/弓/マズルローダー）・精度・反動・リロード速度・腰だめ精度・装弾数・重量・必要スコア・価格・コメント |
| 弾薬・矢 | 名称・種別・有効射程・貫通力・膨張性・重量・必要スコア・価格・適正クラス（最小〜最大）・説明 |
| 銃器-弾薬対応 | どの銃器にどの弾薬が使えるか |
| スコープ | 名称・倍率・重量・必要スコア・価格・説明 |
| スコープ-銃器対応 | どの銃器にどのスコープが使えるか |
| アイテム | 名称・カテゴリ・重量・重量ボーナス（バックパックのみ）・説明 |
| 狩猟区 | 名称・説明 |
| 動物 | 名称・適正狩猟クラス（最小〜最大） |
| 狩猟区-動物対応 | どの狩猟区にどの動物が出没するか |
| シミュレーション | 装備構成・重量・容量・作成日時 |

### 4.2 スキル

現時点で重量に影響するスキルは以下のみ:

| スキル名 | 効果 |
|---|---|
| 荷運びラバ | 基本所持重量を+15%増加（バックパックボーナスには影響なし） |
