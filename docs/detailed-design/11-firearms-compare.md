## 概要

2丁の銃器を並べてスペックを比較する画面。  
重量シミュレータと同様に、銃器一覧からドラッグ&ドロップで比較対象を選択する。

## 機能要件

### 銃器一覧
- 全カテゴリ（ライフル・ショットガン・ハンドガン・弓・マズルローダー）を表示
- 検索ボックスで銃器名をフィルタ
- 各銃器カードを比較欄へドラッグ&ドロップできる
- タップでも比較欄への追加ができる（モバイル対応）

### 比較欄
- スロット2枠固定
- ドロップで銃器をセット、×ボタンで個別に除去
- 同じ銃器を2回選択した場合は追加しない（重複防止）

### 比較表（2丁揃ったとき表示）
比較項目：

| 項目 | フィールド |
|---|---|
| 精度 | accuracy |
| 反動 | recoil |
| リロード速度 | reload_speed |
| 腰だめ精度 | hipfire_accuracy |
| 装弾数 | magazine_capacity |
| 重量 | weight |
| 必要スコア | required_score |
| 価格 | price |

- 数値が高い方（重量・反動・必要スコア・価格は低い方）を緑、劣る方をグレー等で色分け
- 同値の場合は色分けなし

## 技術方針

- **ページ:** `/firearms/compare`（新規）
- **コンポーネント:** `CompareClient`（`"use client"`、@dnd-kit 使用）
- **データ取得:** サーバーコンポーネントで全銃器を取得して渡す
- **DnD実装:** 重量シミュレータ（`SimulatorClient`）の @dnd-kit 実装を参考にする
- **状態管理:** useState で選択中の2丁を管理
